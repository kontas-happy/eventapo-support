# Eventapoへようこそ

Eventapoは、カンファレンスの開催のために生まれた、**イベント管理サービス**です。  
チケット予約、入退場管理、イベント共有をEventapo上で全て行うことができます。

このサイトでは、Eventapoでイベントを開催するための5つの手順について以下の順に説明します。
